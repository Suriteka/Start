# Gulp Recipes
## Installation
Install dependencies :
```
npm i
```
Configure your environment with `.env` file
```
JS_SRC=src/scripts/**/*.js
JS_DEST=assets/scripts/
```

## Run gulp with JavaScript :
Be sure you have `JS_SRC` and `JS_DEST` constants in your `.env` file
```
gulp scripts
```
